# Minor_Project_2
🚀 **Minor Project 2 Completed: SpendDNA – Personal Finance Analytics Using Python & Pandas** 📊💻

I'm excited to share that I have successfully completed my second Data Analytics minor project: **SpendDNA – Personal Finance Analytics Using Python & Pandas**.

In this project, I analyzed **1,328 banking transactions** over a six-month period to understand spending patterns and generate meaningful financial insights.

🔹 **Project Highlights:**

* 📂 Data loading and preprocessing using **Pandas**
* 🧹 Data cleaning and data type conversion
* 📅 Date-based feature engineering (Month, Day, Weekday)
* 🏷️ Transaction categorization (Shopping, Food & Grocery, Transport, Investment, Entertainment, etc.)
* 📊 Monthly spending analysis
* 🛒 Top merchant identification
* 💳 Payment mode analysis (UPI, ATM, NEFT, IMPS)
* 📈 Data visualization using bar charts and pie charts
* 💡 SpendDNA profile based on spending behavior

🛠️ **Tools & Technologies:**

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab / Jupyter Notebook

This project strengthened my understanding of data cleaning, exploratory data analysis (EDA), feature engineering, grouping and aggregation, and data visualization using Python.

I'm continuously improving my skills in **Data Analytics** and **Data Science** by working on practical, real-world projects.
